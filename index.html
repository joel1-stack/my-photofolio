Flagship Project

TrustLayer — My Flagship Build

I'm engineering the trust layer that Africa's informal economy is missing

Why I'm Building This

I started TrustLayer because I kept seeing the same problem — people losing money in transactions where trust doesn't exist. This isn't just a coding project. It's infrastructure I believe East Africa needs.

W
Why I Started
I watched people around me lose money on land deals, vehicle purchases, and freelance payments. Kenya loses KES 10B+ annually to this. Buyers fear paying first, sellers fear delivering first. I realized this trust gap is an engineering problem I can solve.
H
How I'm Solving It
I'm building an API-first escrow layer that any platform can plug into. I've designed it to partner with licensed PSPs to hold funds compliantly — because in fintech, you need a PSP license to hold money, and I've architected the system around that reality. I've also built automated dispute resolution with evidence-based workflows and vendor trust ratings.
W
What I'm Delivering
A REST API that lets any marketplace, freelance platform, or e-commerce site add escrow in under 30 minutes. My universal 5-step flow: Deposit → Hold → Inspect → Release/Refund → Dispute. Simple for developers, safe for users.

⚙ How I Architected the System

Client App→ TrustLayer API→ Licensed PSP→ M-Pesa / Bank

🔒 How I Handle Funds

  • I don't hold funds directly — I partner with a licensed PSP (Payment Service Provider) who manages the escrow wallet under CBK regulation
  • This keeps TrustLayer compliant without needing its own PSP license, and the PSP partnership shapes the entire system architecture
  • TrustLayer orchestrates the full transaction lifecycle via API while the PSP handles the money
  • Revenue model: 1-3% transaction fee split with the PSP partner

⚖ How I Resolve Conflicts

  • I've designed an evidence upload window — photos, documents, chat logs — so both parties can present their case
  • Auto-resolution handles clear-cut cases (delivery tracking confirmations, payment timeouts)
  • Complex disputes escalate to a human mediator with all evidence pre-organized
  • I'm building vendor trust ratings from transaction history — so repeat honest sellers get verified status and faster payouts

💰 What It Costs End Users

  • Buyer pays 1.5% escrow fee on deposit — a small price for peace of mind
  • Seller receives for free — this incentivizes adoption on both sides
  • Platform integrators get free API access up to 100 transactions/month
  • Bottom line: it's always cheaper than losing KES 50K+ to fraud
